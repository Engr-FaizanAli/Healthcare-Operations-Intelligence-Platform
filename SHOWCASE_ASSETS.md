## Upwork Portfolio Entry 1

### Title
OCR-Assisted Lab Result Automation for Diagnostic Workflows

### Description
Built an AI-assisted lab workflow that extracts values from scanned CBC / Complete Blood Picture slips and maps them into structured lab result fields. The solution was designed to reduce repetitive manual entry, improve reporting speed, and support human review before final confirmation.

Key outcomes:
- Scan-to-structured-data workflow for lab reports
- Automatic mapping of extracted values to diagnostic parameters
- Support for derived hematology calculations
- Faster result-entry workflow for laboratory staff

This project reflects my work in document AI, OCR pipelines, and computer vision-driven workflow automation for healthcare environments.

## Upwork Portfolio Entry 3

### Title
End-to-End Healthcare Operations Platform with AI Workflow Integration

### Description
Built a multi-module healthcare operations platform spanning patient flow, lab workflows, pharmacy operations, and financial tracking. A key differentiator was the integration of AI-assisted OCR into clinical data entry, turning scanned diagnostic slips into actionable structured data.

Key outcomes:
- Unified workflows for hospitals, labs, pharmacies, and clinics
- AI-assisted automation embedded into real operational processes
- Reduced manual overhead in diagnostic reporting
- Improved visibility for both frontline users and management teams

This project represents my broader capability in AI engineering, computer vision integration, workflow automation, and domain-specific software design.
